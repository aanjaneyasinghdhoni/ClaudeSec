## GET /api/spans

Returns up to 500 spans ordered by `startNano ASC`.

### Query parameters

| Parameter | Type | Description |
|---|---|---|
| `q` | string | Free text, or `key=value` attribute search |
| `session` | string | Filter by `traceId` |

### Example

```bash
curl "http://localhost:3000/api/spans?q=gen_ai.tool.name=bash&session=trace-abc"
```

Response shape:

```json
{
  "spans": [
    {
      "spanId": "abc123",
      "traceId": "trace-abc",
      "parentId": "claude-code",
      "name": "tool_call/Bash",
      "protocol": "HTTPS",
      "reason": "Processing step",
      "severity": "low",
      "harness": "claude-code",
      "attributes": "{\"gen_ai.tool.name\":\"bash\"}",
      "startNano": "1700000000000000000",
      "endNano": "1700000000500000000"
    }
  ]
}
```

---

## GET /api/spans/:spanId/match

Returns the first matched rule/range metadata for a span.

```json
{
  "spanId": "abc123",
  "severity": "high",
  "matchedLabel": "Prompt injection attempt",
  "matchedText": "ignore previous instructions",
  "matchStart": 184,
  "matchEnd": 213,
  "ruleKey": "builtin-42"
}
```

### Errors

- `404` when the span does not exist
