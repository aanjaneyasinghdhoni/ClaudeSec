---
title: "Alerts API"
description: "Access the immutable threat alert log"
---

## List Alerts

`GET /api/alerts`

<ParamField query="limit" type="number" default="200">
  Maximum number of alerts to return (capped at 1000).
</ParamField>

<ParamField query="severity" type="string">
  Filter by severity: `low`, `medium`, or `high`.
</ParamField>

```json Response
{
  "alerts": [
    {
      "id": 42,
      "ts": "2025-01-01T12:00:00.000Z",
      "ruleLabel": "Recursive root deletion",
      "severity": "high",
      "spanId": "span123",
      "traceId": "trace001",
      "harness": "claude-code",
      "spanName": "bash: rm -rf /tmp",
      "matchedText": "rm -rf /"
    }
  ],
  "total": 42
}
```

## Export Alerts

`GET /api/alerts/export`

Returns all alerts as a downloadable JSON file with `Content-Disposition: attachment`.

## Clear Alerts

`DELETE /api/alerts`

Removes all alerts from the database and emits `alerts-update` via Socket.io.
