2
Universal Agent Harness Support
Auto-detect & ingest telemetry from every major CLI agent: Claude Code, GitHub Copilot, OpenHands, Cursor, Aider, Continue.dev, Cline, and more
Harness detection registry (harnesses.ts)
Typed registry of known agent harnesses with OTLP env var names, default ports, span attribute mappings, and logo assets. Covers: Claude Code, GitHub Copilot CLI, OpenHands, Cursor, Aider, Continue.dev, Cline, Goose, Zed AI, Windsurf.
multi-harness server.ts
Auto-setup CLI wizard (npx claudesec init)
Interactive CLI that detects which agent harness is running, prints exact export commands, and optionally writes to shell profile. Ship as "bin" in package.json.
multi-harness cli/init.ts
Harness identity on spans — badge in UI
Extract resource attributes (service.name, telemetry.sdk.name) to tag each span with its source harness. Show colored harness badge in workflow list and detail panel.
multi-harness UI
Multi-agent topology — separate graph roots per harness
When spans from multiple harnesses arrive, render separate agent root nodes with distinct colors. Show inter-agent edges when one agent calls another via MCP/API.
multi-harness graph
Session isolation — track concurrent agent sessions
Use traceId to group spans into sessions. Show session switcher in sidebar. Each session gets its own graph, timeline, and threat summary.
multi-harness observatory
3
Local Observatory — Timeline & Metrics
Deep visibility into what local agents are doing — timeline view, latency heatmaps, token usage, and span drill-down
Timeline view — Gantt-style span visualization
Second tab alongside the graph: horizontal timeline with spans as bars, colored by severity, grouped by trace. Click bar to open detail panel. Uses startNano/endNano from SQLite.
observatory UI
Metrics panel — tokens, latency, tool-call counts
Aggregate OTel attributes: llm.usage.input_tokens, llm.usage.output_tokens, duration. Show running totals in header stats panel with sparklines (canvas/SVG, no chart lib).
observatory
Span search with attribute filtering
Extend search to filter by any attribute key/value (e.g. tool.name=bash). Add GET /api/spans?q= endpoint with SQLite LIKE query.
observatory API
Persistent sessions — named snapshots
Allow naming and saving snapshots of the current session. Store in separate SQLite table. Load/switch sessions from UI. Add GET/POST /api/sessions endpoints.
observatory
4
Enhanced Security Engine
Richer threat detection, real-time alerts, custom rules, and audit log export
Expand threat rules — prompt injection, exfiltration, supply chain
Add 20+ rules: prompt injection (ignore previous instructions), secret scanning (AWS/GitHub tokens), supply chain (npm from unknown registry), network exfiltration (curl to external IPs).
security
Custom rules UI — add/edit/delete rules in the dashboard
Settings panel where users define custom regex rules with severity, label, description. Persist in rules.json. API: GET/POST/DELETE /api/rules.
security UI
Desktop notifications for HIGH severity spans
Use Web Notifications API to pop native OS notification when HIGH severity span arrives via Socket.io. Works on macOS/Windows/Linux.
security alert
Audit log — immutable append-only threat log
Separate SQLite table "alerts" — every rule match appended with timestamp, rule name, matched text snippet, span context. Export as CSV/JSON from UI.
security
5
Agent Orchestration View
Visualize how multiple agents coordinate, hand off tasks, and call tools — the "big picture" of an agentic workflow
Orchestration graph — agent-to-agent edges
Detect MCP tool calls between agents (gen_ai.tool.call.id correlating to parent span of another agent). Render a separate "Orchestration" tab showing the agent DAG.
orchestration
Sub-agent spawning detection & tree view
Track when Claude Code's Agent tool or similar spawns child agents. Build a collapsible tree showing parent-child agent relationships with per-node threat summary.
orchestration observatory
Tool inventory — what tools each agent uses
Aggregate gen_ai.tool.name attributes across all spans. Show per-agent tool usage heatmap. Flag suspicious tools (bash, exec) with threat counts.
orchestration security
